Addendum

As a big fan of folklore in the game Final Fantasy XIV, it's very excited for me to see a new typeface in use in the new city Solution Nine introduced in Dawntrail.

I've previously developed other fonts (feel free to check my GitHub!) But it's really my first time developing a variable font from scratch. It's super fun but also challenging at the same time. (Next time I'll know to develop the masters from the extreme point instead of from the middle.)

NeonSky is developed in Glyphs. It has five masters (Thin, Thin Ultra Expanded, Medium, Medium Ultra Expanded and Black) and 22 instances as static fonts generated via interpolation. If you are using a software that support variable font, I highly recommend using the VF version of the font.

Also it's really my first time developing a web page. I learned a lot of HTML/CSS and even some JS code from scratch, thanks to stackoverflow, CSS-Tricks, MDN, and last but not least, designers and developers of League Mono. I feel very shameful that I borrowed a lot of design and script snipet from the demo page of League Mono. I would not be able to make this demo page without this reference.

I'm also happy to release the font under the SIL Open Font License v1.1, with Reserved Font Name NeonSky. Feel free to give me feedback, and I would appreciate a Ko-Fi if you want to :)
